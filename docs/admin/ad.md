## 接口说明


#### 1.0 添加广告

##### 接口说明

添加广告

##### 请求说明

| http 请求方式          | post     |
|:------------- |:---------------:|
| url      | /admin/ad/create |

#####  输入参数


```json  

{
  "content": "string",  //描述
  "deleted": true, //是否删除
  "enabled": true, //是否开启
  "extParams": {},  //附加参数
  "idValue": 0, //跳转ID
  "link": "string", //跳转链接（H5）
  "name": "string", //标题
  "position": 0,  // 1 首页 2 活动 3 用户 
  "type": 0, //跳转类型  0:H5 1:产品详情 2:活动
  "url": "string" //图片地址 //图片地址
}

```

#####  错误说明




#####  返回实例

```json
    "c": 0,
    "m": null,
    "d": {
    }
}  
```

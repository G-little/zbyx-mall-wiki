#### 1.1 酿化续费接口

##### 接口说明

酿化续费

##### 请求说明

| http 请求方式          | post |
|:------------- |:---------------:|
| url      | /aging/pay |

#####  输入参数


| 参数          |必选             | 类型       | 参数说明        | 备注          |
|:-------------|:---------------:|:-------------|:-------------|:-------------|
| orderId      | 是 |  int  |  订单ID |  |
| year      | 否 |  int  |  续费周期 |  不填系统会根据最短续费时间计算，如果用户选择则安选择的值  |
| payType      | 是 |  string  |   支付方式 | 根据结算页拉取的支付项填写，小程序可以固定写死 WX_MP  |




#####  错误说明




#####  返回实例
```json
    
{
    {
    "c": 0,
    "m": null,
    "d": {
        
    }
}

```

#### 1.0 搜索主页

##### 接口说明

获取搜索主页信息

##### 请求说明

| http 请求方式          | get     |
|:------------- |:---------------:|
| url      | /search/index |

#####  输入参数

无

#####  错误说明


#####  返回实例
```json
    
    {
    "c": 0,
    "m": null,
    "d": {
        "defaultKeyword": {  //默认搜索词
            "id": 6,  //关键词ID
            "userId": null, //用户ID
            "keyword": "520元礼包抢先领", //关键词
            "from": null //来源
        },
        "historyKeywordList": null, //历史搜索词
        "hotKeywordList": [ //热门搜索词
            {
                "id": 6,
                "userId": null,
                "keyword": "520元礼包抢先领",
                "from": null
            }
        ]
    }
}

```

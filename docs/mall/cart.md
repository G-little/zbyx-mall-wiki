##### 接口说明

获取购物车商品总数

##### 请求说明

| http 请求方式          | get     |
|:------------- |:---------------:|
| url      | /cart/goodscount |

#####  输入参数

无

#####  错误说明


#####  返回实例
```json
   
   {
    "c": 0,
    "m": null,
    "d": 0  //商品总数
} 
   
```

接口说明

1 新闻

1.1 新闻列表

接口说明

分页拉取新闻列表

请求说明
http 请求方式 get
url {{host}}/www/news/list
token说明 无
输入参数
参数 必选 类型 参数说明 备注
page 否 int 分页 默认值1
limit 否 int 分页条数 默认值10
错误说明



返回实例
{
    "c": 0,
    "m": null,
    "d": {
        "pageSize": 10,
        "currentPage": 1,
        "list": [
            {
                "id": 1,
                "title": "test",
                "brief": "test",
                "addTime": null,
                "deleted": false,
                "enabled": false,
                "content": null
            }
        ],
        "end": true,
        "empty": false,
        "startIndex": 0,
        "totalPage": 1
    }
}

接口说明

1. 商户管理

字段说明

字段 类型 说明 备注
id int 商户ID
name string 商户名称
addr string 商户地址
logoUrl string 头像
mobile string 手机号
type byte 商户类型 0 默认 1

1.1.0 商户列表

接口说明
请求说明
http 请求方式 get
url /admin/agent/list
输入参数
参数 必选 类型 参数说明 备注
page 否 int 分页
limit 否 int 单页条数
name 否 string 名字
id 否 int 代理商名字
mobile 否 string 手机号
type 否 string 类型
错误说明
返回实例

{
    "c": 0,
    "m": null,
    "d": {
        "pageSize": 10,
        "currentPage": 1,
        "list": [
            {
                "id": 1,  //商户ID
                "name": "xiaogang", //商户名称
                "addr": "wangjing", //商户地址
                "logoUrl": "/logo", // 商户Logo
                "deleted": false, // 删除状态
                "addTime": null, //添加时间
                "mobile": null //手机号
            }
        ],
        "end": true,
        "startIndex": 0,
        "totalPage": 1,
        "empty": false
    }
}
